The "ATTACK OF THE GIANT SNAKE!" movie is in AVI format (2.11MB zipped).

You will need an unzipper to unzip the movie. If you need one, click here to download FreeZip (not the best, but it's small and will do the job =)

You will also need an AVI player, if you don't have one (if you're not sure, download and unzip the file and double click on it, if nothing happens or something "wrong" happens, then you probably need an AVI player). Download the Windows Media Player from Microsoft, here.

And... click here to download the movie! =) Enjoy!

Step-by-step instructions... (for you non-computer-nerd people =) (These instructions are assuming that you need to download ALL of the above programs/files)
1. Download the movie file
2. Download FreeZip
3. Go to Microsoft's Windows Media Player site and download the Media Player.
4. Install the Media Player and FreeZip (double click on files)
5. Right click on the movie.zip file and go to UnZip.
6. You should see a "The Attack of the Giant Snake!.avi" file, double click on it and it should play =)
